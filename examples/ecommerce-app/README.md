# E-Commerce App Example

This is an example app generated by OmniForge based on the idea:

> "An e-commerce platform with product catalog, shopping cart, checkout, and order management"

## Features

- Product catalog with search
- Shopping cart
- Checkout with Stripe integration
- Order management
- User authentication
- Admin dashboard

## Generated Structure

- Frontend: Next.js with product pages, cart, checkout
- Backend: NestJS API with payment processing
- Database: PostgreSQL schema for products, orders, users
- Integrations: Stripe for payments
- Authentication: Clerk for user management

## Getting Started

```bash
npm install
npm run db:migrate
npm run dev
```

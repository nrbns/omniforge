# Telemedicine App Example

This is an example app generated by OmniForge based on the idea:

> "A telemedicine app for remote consultations with video calls, prescription management, and appointment scheduling"

## Features

- Video consultations
- Patient dashboard
- Doctor dashboard
- Appointment scheduling
- Prescription management
- Medical records

## Generated Structure

- Frontend: Next.js app with React components
- Backend: NestJS API with Prisma ORM
- Database: PostgreSQL schema for patients, doctors, appointments
- Real-time: WebSocket for video call signaling
- Authentication: Clerk integration

## Getting Started

```bash
# Install dependencies
npm install

# Run database migrations
npm run db:migrate

# Start development server
npm run dev
```

## API Endpoints

- `GET /api/patients` - List patients
- `GET /api/doctors` - List doctors
- `POST /api/appointments` - Create appointment
- `GET /api/appointments/:id` - Get appointment
- `POST /api/prescriptions` - Create prescription

## Environment Variables

```env
DATABASE_URL=postgresql://...
NEXT_PUBLIC_API_URL=http://localhost:3001/api
CLERK_PUBLISHABLE_KEY=...
```
